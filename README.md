# 數位工廠營運中心協作所

以統一命名空間（Unified Namespace）架構整合全廠即時數據，提供集中式營運監控與決策之中樞平台。

- 原始系統：數位工廠營運中心（Digital Factory Operations Center / UNS平台）
- 產業分類：智慧製造｜製造管理
- 使用部門：廠務主管、資訊部門、製造工程、高階管理層
- 工作流程：現場數據發布至UNS → 標準化命名整合 → 中央平台訂閱應用 → 即時監控呈現 → 異常事件通知決策
- AI 應用：AI基於統一數據源建立跨系統預測模型、智慧產出全廠營運異常摘要
- 主要 KPI：數據整合覆蓋率、營運異常反應時間、跨系統查詢效率
- 代表產品：HighByte Intelligence Hub、Ignition（Inductive Automation）UNS架構、Litmus Automation
